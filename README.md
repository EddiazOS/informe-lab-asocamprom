# Asocamprom

Informe microbiológico en Typst para chorizo vegano a base de lentejas.

## Estructura

- `main.typ`: documento principal del informe.
- `docs/referencias.bib`: bibliografía técnica y normativa.
- `docs/imagenes/.gitkeep`: carpeta reservada para evidencias fotográficas.

## Nota técnica

Este repositorio reutiliza únicamente la arquitectura documental de informes anteriores. No debe copiar resultados, discusión ni conclusiones entre matrices distintas.
